ラブライブ！用
コール&レスポンスとか、公式サイトのヘッドラインとか。

# Usage
```
lovelive = new LoveLive()

lovelive.call("穂乃果") # ほのか|ﾎﾉｶﾁｬﾝ|高坂穂乃果
# => ファイトだよ！
lovelive.call("ファイトだよ！") # ファイトだよ
# => うんっ！ or ファイトだよ！じゃねーっつーの！！(1%)

lovelive.news() # 公式サイトのヘッドライン
```
